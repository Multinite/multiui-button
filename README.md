# Button

A button component for [MultiUI](https://github.com/ping/multiui).

## Usage

```js
import Button from "@/components/multiui/Button";

<Button>Click me</Button>;
```

## Props

| Name     | Type      | Default | Description           |
| -------- | --------- | ------- | --------------------- |
| children | ReactNode |         | The button's content. |
